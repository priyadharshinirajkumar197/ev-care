# EV Care

EV Care is a UX-focused EV health companion designed to help non-technical electric vehicle owners understand battery health, charging habits, maintenance needs, and in-vehicle infotainment through clear and intuitive design.

## Focus
- Simplifying complex EV and vehicle system information
- Reducing user anxiety with clarity-first UX
- Designing a seamless vehicle + infotainment experience

## Key Features
- Visual battery health overview
- Charging and maintenance insights
- Vehicle health summaries without technical jargon
- Infotainment dashboard for media and in-car interactions

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS
- Framer Motion

## Role
Product concept, UX & interaction design, frontend implementation.

## Status
Independent portfolio project.
